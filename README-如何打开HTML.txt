  如何打开 HTML 文件
═══════════════════════════════════════════════════════

⚠️ 重要提示：
   不能直接双击 index.html 文件！
   直接打开会有 CORS 错误，游戏无法运行。

✅ 正确的打开方式：

方法 1：双击批处理文件（最简单）⭐ 推荐
─────────────────────────────────────────────────────
   → 双击 "双击打开HTML.bat"
   → 游戏会自动在浏览器中打开
   → 使用本地服务器，游戏可以正常运行

方法 2：使用预览命令
─────────────────────────────────────────────────────
   1. 打开命令提示符或 PowerShell
   2. 进入项目目录
   3. 运行：npm run preview
   4. 浏览器访问：http://localhost:4173

方法 3：使用开发服务器
─────────────────────────────────────────────────────
   → 双击 "start-game.bat"
   → 游戏会在 http://localhost:5173 打开
   → 这是开发模式，支持热重载

方法 4：使用简单服务器
─────────────────────────────────────────────────────
   → 双击 "简单服务器.bat"
   → 使用 Python 或 http-server
   → 游戏会在 http://localhost:8080 打开

═══════════════════════════════════════════════════════
为什么不能直接打开 HTML？
═══════════════════════════════════════════════════════

直接打开 HTML 文件时：
✗ 浏览器使用 file:// 协议
✗ 浏览器安全策略阻止加载 JavaScript 模块
✗ 会出现 CORS（跨域）错误
✗ 游戏无法正常运行

必须使用 HTTP 服务器：
✓ 本地服务器提供 HTTP 协议
✓ 可以正常加载 JavaScript 模块
✓ 游戏可以正常运行

═══════════════════════════════════════════════════════
推荐使用方式
═══════════════════════════════════════════════════════

最简单：双击 "双击打开HTML.bat"
   - 自动构建（如果需要）
   - 自动启动服务器
   - 自动打开浏览器
   - 一键完成所有操作
