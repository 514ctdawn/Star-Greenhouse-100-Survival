======================================== 
Star Greenhouse: 100% Survival 
Portable Version 
======================================== 
 
How to run: 
 
1. Double-click "START-GAME.bat" 
   - Requires Python installed 
   - Game will open in browser 
 
2. Manual: 
   - Open command prompt 
   - Go to dist folder 
   - Run: python -m http.server 8000 
   - Visit: http://localhost:8000
