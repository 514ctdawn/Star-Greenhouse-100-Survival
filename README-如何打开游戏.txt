  Star Greenhouse: 100% Survival - 如何打开游戏
═══════════════════════════════════════════════════════

⚠️ 重要提示：
   不要直接双击 index.html 文件！
   直接打开 HTML 文件会有错误（CORS 错误）。

✅ 正确的打开方式：

方法 1：双击批处理文件（最简单）
   → 双击 "双击打开游戏.bat" 文件
   → 游戏会自动在 Chrome 浏览器中打开

方法 2：双击其他启动文件
   → "打开游戏.bat" - 可以选择在线版本或本地版本
   → "在Chrome中打开.bat" - 直接在 Chrome 中打开
   → "打开网站.bat" - 使用默认浏览器打开

方法 3：手动访问链接
   在浏览器地址栏输入：
   https://514ctdawn.github.io/Star-Greenhouse-100-Survival/

方法 4：本地开发服务器
   双击 "start-game.bat" 文件
   游戏会在 http://localhost:5173 打开

═══════════════════════════════════════════════════════
为什么不能直接打开 index.html？
═══════════════════════════════════════════════════════

直接打开 HTML 文件时：
✗ 浏览器使用 file:// 协议
✗ 浏览器安全策略会阻止加载 JavaScript 模块
✗ 会出现 CORS 错误

必须使用 HTTP/HTTPS 协议：
✓ GitHub Pages 提供 HTTPS 协议
✓ 本地开发服务器提供 HTTP 协议
✓ 这样才能正常加载 JavaScript 模块
