═══════════════════════════════════════════════════════
  如何打包项目并发送给其他人
═══════════════════════════════════════════════════════

步骤 1：创建便携版
═══════════════════════════════════════════════════════

1. 在项目根目录运行：
   双击：create-portable-simple.bat

2. 等待创建完成
   - 会自动构建项目
   - 创建 Star-Greenhouse-Portable 文件夹
   - 包含所有必要文件

步骤 2：测试便携版
═══════════════════════════════════════════════════════

1. 进入 Star-Greenhouse-Portable 文件夹
2. 双击 START-GAME.bat
3. 确认游戏可以正常运行
4. 如果正常，继续下一步

步骤 3：打包成 ZIP
═══════════════════════════════════════════════════════

方法 A：使用 Windows 资源管理器
  1. 右键点击 Star-Greenhouse-Portable 文件夹
  2. 选择"发送到" → "压缩(zipped)文件夹"
  3. 重命名 ZIP 文件（可选）

方法 B：使用压缩软件
  1. 使用 WinRAR、7-Zip 等软件
  2. 选择 Star-Greenhouse-Portable 文件夹
  3. 压缩成 ZIP 格式

步骤 4：发送 ZIP 文件
═══════════════════════════════════════════════════════

可以通过以下方式发送：
  - 电子邮件附件
  - 云盘（Google Drive、OneDrive、Dropbox）
  - 即时通讯软件（微信、QQ、Telegram）
  - USB 闪存盘

步骤 5：接收者操作
═══════════════════════════════════════════════════════

接收者需要：
  1. 解压 ZIP 文件
  2. 进入 Star-Greenhouse-Portable 文件夹
  3. 双击 START-GAME.bat
  4. 游戏自动在浏览器中打开

═══════════════════════════════════════════════════════
📋 完整操作流程
═══════════════════════════════════════════════════════

开发者（你）：
  [1] 运行 create-portable-simple.bat
  [2] 测试 Star-Greenhouse-Portable 文件夹
  [3] 压缩成 ZIP 文件
  [4] 发送给其他人

接收者：
  [1] 解压 ZIP 文件
  [2] 双击 START-GAME.bat
  [3] 游戏自动运行

═══════════════════════════════════════════════════════
✅ 检查清单
═══════════════════════════════════════════════════════

打包前检查：
  [ ] Star-Greenhouse-Portable 文件夹存在
  [ ] dist 文件夹包含所有文件
  [ ] START-GAME.bat 存在
  [ ] README.txt 存在
  [ ] 测试运行正常

发送前检查：
  [ ] ZIP 文件可以正常解压
  [ ] 解压后文件完整
  [ ] 包含使用说明

═══════════════════════════════════════════════════════
💡 提示
═══════════════════════════════════════════════════════

- ZIP 文件大小约 300-500 KB（仅构建文件）
- 如果包含源代码，文件会更大（约 50-100 MB）
- 推荐只发送便携版（Star-Greenhouse-Portable）
- 接收者需要 Python 或 Node.js（脚本会自动检测）
