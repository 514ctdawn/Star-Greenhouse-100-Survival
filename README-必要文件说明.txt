保留的必要文件：

开发使用：
  - start-game.bat          # 启动开发服务器
  - create-portable-simple.bat  # 创建便携版

便携版（Star-Greenhouse-Portable/）：
  - START-GAME.bat          # Python 启动脚本（推荐）
  - START-GAME-NODE.bat     # Node.js 启动脚本（备用）

═══════════════════════════════════════════════════════
如何使用
═══════════════════════════════════════════════════════

本地开发：
  双击：start-game.bat
  访问：http://localhost:5173

创建便携版：
  双击：create-portable-simple.bat
  等待创建完成

打包发送：
  1. 进入 Star-Greenhouse-Portable 文件夹
  2. 压缩成 ZIP
  3. 发送给其他人

接收者使用：
  1. 解压 ZIP 文件
  2. 双击 START-GAME.bat
  3. 游戏自动运行
